# Blinkit Business Analysis

## Project Overview
This project analyzes Blinkit sales data using Microsoft Excel and MySQL. The goal is to gain business insights and create an interactive dashboard.

## Tools Used
- Microsoft Excel
- MySQL
- Pivot Tables
- Pivot Charts
- Slicers

## Project Workflow
1. Cleaned the dataset in Excel.
2. Imported the dataset into MySQL.
3. Performed SQL queries for analysis.
4. Created an interactive Excel dashboard.

## Key KPIs
- Total Sales: $18.6M
- Average Sales: $2.2K
- Number of Items: 8523
- Average Rating: 2.6
